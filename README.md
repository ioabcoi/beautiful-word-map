# 🗺️ Beautiful-Word-Map 기획서

> 전 세계 언어 속에 숨겨진 보물 같은 단어들을 찾아 나만의 지도를 그려가는 기록

---

## 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 프로젝트명 | Beautiful-Word-Map (뷰티풀 워드 맵) |
| 유형 | 단일 페이지 SPA (React) |
| 분위기 | 따뜻하고 아날로그 감성 (크림, 베이지) |
| 목적 | 전 세계 언어 속 특별한 단어들을 수집하고 언어별·테마별로 탐색하는 개인 사전 |

---

## 메뉴 구조

```
헤더
 ├── 타이틀: Beautiful Word Map
 └── 필터 탭: [전체] [언어별 ▾] [테마별 ▾]

메인 콘텐츠
 └── 단어 카드 그리드

플로팅 버튼 (우측 하단)
 └── [+ 단어 추가]
```

---

## 화면 흐름

```
① 기본 화면
   전체 단어 카드 그리드 (최신순)

② 언어 필터 선택
   일본어 / 한국어 / 포르투갈어 / 독일어 ...
   → 해당 언어 카드만 표시

③ 테마 필터 선택
   테마 목록 드롭다운 (직접 생성 가능)
   → 해당 테마 카드만 표시

④ 단어 카드 클릭
   → 카드 확장 (제자리에서 펼쳐지기)
      발음 · 의미 · 예시 · 테마 태그 표시

⑤ + 버튼 클릭
   → 슬라이드업 폼
      (언어 / 단어 / 발음 / 의미 / 예시 / 테마 선택)
```

---

## 단어 카드 구성

```
┌─────────────────────────┐
│  🇯🇵  일본어              │  ← 언어 플래그 + 이름
│                         │
│  木漏れ日               │  ← 단어 (크게)
│  코모레비               │  ← 발음
│                         │
│  나뭇잎 사이로 새어드는  │  ← 의미 (짧게)
│  빛의 반짝임            │
│                         │
│  🌿 빛이 머무는 찰나    │  ← 테마 태그
└─────────────────────────┘
```

**클릭 시 확장 (제자리에서 펼쳐지기)**
- 예시 문장 추가 표시
- 수정 / 삭제 버튼 노출

---

## 기능 정의

### 단어 관리
- 추가 — 플로팅 + 버튼 → 슬라이드업 폼
- 수정 — 카드 확장 시 수정 버튼
- 삭제 — 카드 확장 시 삭제 버튼

### 필터
- 언어별 필터 (드롭다운)
- 테마별 필터 (드롭다운)
- 테마는 직접 생성 가능

### 테마 예시
| 테마명 | 설명 |
|--------|------|
| 🌿 빛이 머무는 찰나 | 자연이 우연히 선물해준 반짝임과 시각적인 아름다움 |
| 🌊 말로 못한 감정 | 딱 맞는 단어가 없어 담아두었던 복잡한 감정들 |
| 🕰️ 시간의 결 | 흘러가는 시간과 기억, 그리고 덧없음에 대하여 |

## 데이터 구조

```js
// 단어
{
  id,
  language,   // "일본어"
  flag,       // "🇯🇵"
  word,       // "木漏れ日"
  pronunciation, // "코모레비"
  meaning,    // "나뭇잎 사이로 새어드는 빛의 반짝임"
  example,    // 예시 문장
  themeId     // 테마 id 참조
}

// 테마
{
  id,
  emoji,
  name,
  description
}
```
